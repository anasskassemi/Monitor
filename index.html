Hand Gesture-Based Mouse Control

By Anass Kassemi

This project enables controlling a computer mouse using hand gestures via webcam input, leveraging MediaPipe and OpenCV.

🎯 Features

  • Real-time hand tracking via webcam
  • Smooth mouse movement using linear interpolation (LERP)
  • Gesture recognition:
    • Index finger up → Mouse move
    • Index only → Click
    • Two fingers → Scroll
    • Three fingers → Right-click
    • One non-index finger → Left-click

📦 Requirements

Install dependencies using:

pip install -r requirements.txt

🚀 How to Run

bash run.sh

or directly:

python script.py

📁 Project Files

  • script.py – Main Python script for gesture recognition and mouse control.
  • requirements.txt – Python dependencies list.
  • run.sh – Shell script to launch the project.
  • index.html – Optional UI or project description placeholder.
